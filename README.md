## General info
This project is simply based on collecting information of an employee.

## Technologies
Project is created with:
* Rails 7
* Stimulus js
* Hotwire
* Tailwind
* TurboFrames 
* MySQL 
* RSpec 

## Setup
To run this project, install it locally using npm:

```
$ bundle install
$ db:create
$ db:migrate
$ ./bin/dev
```


## RSpec
To run specs:

```
$ rails spec
```
